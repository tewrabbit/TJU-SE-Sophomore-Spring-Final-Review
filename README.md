##  简介

本仓库整理了 **2026 年同济大学软件工程大二下学期** 的期末复习资料，包括 5 门核心课程：操作系统，数据库，算法，系统结构，计网。内容包括：

- 各科课件整理
- 往届学长学姐的学习笔记与复习资料
- 期末真题与练习题
- Claude 辅助整理的复习 Markdown（仅供参考，主体复习请以教材和其他资料为准）

希望能帮到自己，也帮到大家
