## 三、各题型的应试技巧

### 填空题（10 分，每空 0.5 分）

**特点**：考的是死记硬背的知识点
**常考内容**：
- OSI 七层的名称、PDU 名称
- 各种协议属于哪一层
- 特殊数字（64 字节、48 位、1.544 Mbps 等）
- 协议缩写全称
- 特殊 IP 地址

**应对**：背 Quick-Reference-Cheat-Sheet.md 里面的 "Key Numbers" 和 "Special IP Addresses"

### 判断题（对错题）

**特点**：考的是概念性理解，尤其是容易混淆的点
**常考内容**：
- "A 等于 B" 类（比如 "波特率等于比特率" → 错！）
- "X 属于 Y 层" 类（比如 "ARP 属于传输层" → 错！ARP 是网络层）
- "X 设备能做 Y" 类（比如 "交换机可以隔离广播域" → 错！路由器才能）

**应对**：看 Quick-Reference-Cheat-Sheet.md 里面的 "Common Pitfalls"

### 单选题（40 分，每题 2 分）

**特点**：覆盖面广，但通常有 2 个明显错误选项
**应对**：
1. 排除法先去掉明显错的
2. 注意题目中的关键词（"which LAYER", "maximum", "minimum" 等）

### 大题（50 分，6 题）

**特点**：计算 + 画图 + 分析
**应对**：
1. 先做自己会的，难的留到最后
2. 公式写对了就有分（步骤分），不要只写答案
3. 画图题标注清楚横纵坐标
4. 单位别忘写（bps, ms, bytes 等）

### PPT 上的细节怎么处理？

老师说了 "细节都在 PPT 里"。这些细节主要是：
1. **具体的数值**：比如 SIFS = 28 μs, DIFS = 128 μs
2. **具体的人名/协议名**：比如 HDLC 的三种模式 NRM/ARM/ABM
3. **具体的步骤**：比如 CSMA/CA 的完整流程
4. **某张具体的图**：比如交换机的 MAC 地址学习过程

**怎么复习这些细节？**
→ 我整理的 Ch0x 文档已经把 PPT 中的细节全部提取并写进去了。你不需要自己从 PPT 中找细节，看我的文档就够了。如果某个细节我漏了，你可以在真题中发现，然后补上。

## 六、知识点重要程度分级

### ★★★（必考，必须完全掌握）

- OSI 七层模型和 TCP/IP 四层模型
- Nyquist 和 Shannon 公式计算
- CSMA/CD 最小帧长计算
- 滑动窗口：GBN/SR 窗口大小约束
- 帧序号字段长度设计
- IP 分片与重组
- 特殊 IP 地址
- TCP 三次握手和四次挥手
- TCP 拥塞控制（慢开始/拥塞避免/快重传/快恢复）
- RIP/OSPF/BGP 对比

### ★★（高频，应该掌握）

- 交换技术延迟对比
- CRC 计算
- CDMA 解码
- T1 载波结构
- 交换机 MAC 地址学习
- CSMA/CA 的 RTS/CTS
- CIDR 子网划分
- NAT 工作原理
- ARP 工作过程
- M/M/1 排队模型
- 漏桶/令牌桶计算
- WFQ 分组调度

### ★（可能考，了解即可）

- ALOHA 吞吐量推导
- 自适应树遍历
- 千兆以太网设计
- 广播路由分组计数
- 隧道技术
- RSVP
- 移动主机三角路由
- IPv6
