## 二、考试基本信息回顾

```
题型分布：
  选择题 20道 × 1分 = 20分（基础概念为主）
  填空题 10道 × 1分 = 10分（关键术语和数字）
  判断题 10道 × 1分 = 10分（概念辨析，陷阱多）
  简答题  4道 × 5分 = 20分（概念阐述）
  大题    4道 × ?分 = 40分（计算和设计）

大题考点（来自题型总结.txt + 真题分析）：
  ① P/V操作（信号量同步设计）
  ② 进程调度算法（FCFS/SJF/RR等时间计算）
  ③ 页式机制（地址转换 + 页面置换算法）
  ④ 多级索引 / 目录改进 / 磁盘调度（文件系统计算）
  ⑤ 银行家算法（死锁避免，近年来常考）
  ⑥ 段页式地址转换
```

## 四、第一阶段：基础构建（建议3-5天）

**目标**：建立完整的知识框架，知道每个模块讲什么，核心概念能说出来。

### Day 1：概述 + 进程基础

**第一步（5分钟）**：先打开 `题型与考试信息总结.txt` 看一眼，心里有数考什么。

**第二步（30分钟）**：打开PPT `课堂ppt/ch1 概述.ppt` 和 `课堂ppt/ch4 进程基础.ppt`，快速浏览。重点看标题和加粗内容，细节不用记。

**怎么读**：不要死记。每读完一个大标题下的内容，盖住屏幕，用自己话复述一遍核心意思。

**第四步（15分钟）**：打开 `02-核心概念速查表.md`，找到对应章节的表格行，做一遍"看概念遮定义"的自我测试。

**睡前（10分钟）**：再打开 `02-核心概念速查表.md`，快速过一遍今天学过的概念。

## 六、第三阶段：刷题实战（建议3-4天）

**目标**：从"知道"到"做对"，建立做题手感，发现知识盲区。

### 刷题总策略

```
先分章刷（按模块巩固）→ 后综合刷（模拟考试）
先做小题（选择/填空/判断）→ 后做大题（计算/设计）
错题必须标记 → 回头查知识点 → 重做到对为止
```

**Day 2：内存管理**

1. 打开 `操作系统期末考试.pdf`（2019-2020真题），**限时120分钟完整做一遍**
2. 对答案，标记所有错题
3. 分析每道错题的原因：
   - 概念不清 → 回去翻 File 1 对应章节
   - 计算错误 → 重做并整理计算步骤
   - 审题失误 → 标记陷阱类型
4. 把错题涉及的知识点在 File 1 中找到原文，重新理解一遍

### 刷题时的具体技巧

**选择题**：
- 每个选项都要知道为什么对、为什么错
- 特别注意"不是"、"错误的是"这种否定问法
- 排除法是保底手段，但最好能直接选出正确答案

**填空题**：
- 关注 File 5 中列出的所有"常考数字记忆"条目
- 各种"最"字题（最基本、最常用、最好的方法……）

**判断题**：
- 关注 File 5 中列出的"常见判断题陷阱"
- 注意绝对化表述（"一定"、"必须"、"都"）往往是错的
- 仔细读每个字，不要跳读

**简答题**：
- 不需要背全文，记住**关键词和逻辑链**即可
- 分点作答，每点一句话
- 例如"什么是虚拟存储器？"→ 定义(1句) + 三大特征 + 容量确定 + 局部性原理基础

**大题**：
- P/V操作：先分析进程角色 → 确定信号量及初值 → 写出伪代码 → **检查是否可能死锁**
- 调度计算：画出Gantt图（时间线）→ 计算各进程完成时间 → 代入公式
- 地址转换：先判断页号是否在内存（有无缺页）→ 再用公式算物理地址
- 文件索引：分清直接/一级间接/二级间接 → 计算每块能放几个索引项 → 乘块大小

## 十一、常见问题

**Q: PPT和File 1有冲突怎么办？**
A: 以File 1为准（因为File 1综合了PPT、学长笔记、复习题等所有资料）。如果发现明显不一致，再去查PPT原文确认。

**Q: 感觉内容太多记不住怎么办？**
A: 不需要全部背下来。分三层：
- **必须记住的**（核心定义、公式、算法步骤、数字答案）——靠重复
- **需要理解的**（算法原理、概念之间的关系）——靠用自己的话复述
- **了解就行的**（细节、历史背景）——看一遍有印象即可

**Q: 大题做不对怎么办？**
A: File 4 的每道例题都有**完整的解题步骤**。先跟着步骤走一遍理解思路，然后遮住答案自己从头算。如果中间卡住了，就看一眼卡住的那一步，然后继续自己算。重复2-3次就能独立完成。

**Q: 时间不够，应该优先放弃哪个模块？**
A: 不建议完全放弃任何模块。但如果必须压缩：
- 设备管理可以只看File 1第六部分和File 5对应内容（概念为主，考得少）
- 体系结构可以只看File 2表格（几个关键概念就够了）
- **进程管理绝对不能放弃**，35%的分值
