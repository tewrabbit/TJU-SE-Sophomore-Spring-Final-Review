| 英文 | 中文 | 定义/要点 |
|------|------|------|

| Closure (α⁺) | 属性闭包 | 由α能推出的所有属性 |
| Armstrong's Axioms | 阿姆斯特朗公理 | 推导FD的三条基本规则 |
| Reflexivity | 自反律 | β⊆α ⇒ α→β |
| Augmentation | 增补律 | α→β ⇒ αγ→βγ |
| Transitivity | 传递律 | α→β, β→γ ⇒ α→γ |
| Union Rule | 合并律 | α→β, α→γ ⇒ α→βγ |
| Decomposition Rule | 分解律 | α→βγ ⇒ α→β, α→γ |
| Pseudotransitivity | 伪传递律 | α→β, γβ→δ ⇒ αγ→δ |
| Extraneous Attribute | 无关属性 | 去掉后F⁺不变 |
| Canonical Cover (Fc) | 正则覆盖 | 最小等价的FD集合 |

## 十、简答题常用英文表达

### 开头模板
- **定义概念**: "XXX refers to ..." / "XXX is defined as ..."
- **列举**: "There are N main types of XXX: ..." / "The key features include: ..."
