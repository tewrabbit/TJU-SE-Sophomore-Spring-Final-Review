## 二、全部核心公式速查

### 2.1 第1章 — 定量分析

#### Amdahl定律

$$Speedup = \frac{T_{before}}{T_{after}} = \frac{1}{(1-F_e) + \frac{F_e}{S_e}}$$

- $F_e$：可改进部分占总执行时间的比例
- $S_e$：改进部分的加速比
- **推论**：即使 $S_e \to \infty$，最大加速比 = $\frac{1}{1-F_e}$

#### CPU性能公式

$$CPU\ Time = IC \times CPI \times T_c$$

- $IC$（Instruction Count）：执行的指令总数
- $CPI$（Cycles Per Instruction）：平均每条指令的时钟周期数
- $T_c$（Clock Cycle Time）：时钟周期时间 = $1/f$

$$CPI = \frac{\sum_{i=1}^{n}(CPI_i \times IC_i)}{IC} = \sum_{i=1}^{n}\left(CPI_i \times \frac{IC_i}{IC}\right)$$

#### 程序局部性原理

- **时间局部性**：即将用到的信息很可能就是目前正在使用的信息
- **空间局部性**：即将用到的信息很可能与目前正在使用的信息在空间上相邻

---

### 2.2 第3章 — 流水线技术

#### 吞吐率 TP

$$TP = \frac{n}{T_k} = \frac{n}{(k + n - 1)\Delta t}$$

- $n$：任务数，$k$：流水线段数，$\Delta t$：时钟周期
- 最大吞吐率：$TP_{max} = \frac{1}{\max\{\Delta t_i\}}$（由瓶颈段决定）

#### 加速比 S

$$S = \frac{T_{顺序}}{T_{流水}} = \frac{n \cdot k \cdot \Delta t}{(k + n - 1)\Delta t} = \frac{nk}{k + n - 1}$$

- 最大加速比 = $k$（当 $n \to \infty$）

#### 效率 E

$$E = \frac{n个任务实际占用的时空区}{k个段总的时空区} = \frac{n}{k + n - 1}$$

$$E = \frac{S}{k} = TP \cdot \Delta t$$

---

### 2.3 第4章 — 指令级并行

#### IPC与CPI关系

$$IPC = \frac{1}{CPI}$$

$$CPI_{实际} = CPI_{理想} + Stall_{结构} + Stall_{数据} + Stall_{控制}$$

#### 向量链接时间

向量执行时间 = 启动时间 + 向量长度

链接时：$T = T_{startup} + N$（$T_{startup}$ 为链接建立时间，约等于各部件流水深度之和）

具体计算见 File 4。

### 2.4 第5章 — 存储系统

#### 两级存储层次

$$T_A = T_1 + F \times T_M$$

- $T_A$：平均访问时间
- $T_1$：M1（靠近CPU的那级）的访问时间
- $F$：不命中率（失效率）= $1 - H$
- $T_M$：不命中开销 = $T_2 + T_B$（M2访问时间 + 块传送时间）

#### Cache 平均访存时间 (AMAT)

$$AMAT = Hit\ Time + Miss\ Rate \times Miss\ Penalty$$

#### 带Cache的CPU时间

$$CPU\ Time = IC \times \left(CPI_{exec} + \frac{MemAccess}{Instr} \times MissRate \times MissPenalty\right) \times ClockCycle$$

另一种等价形式：

$$CPU\ Time = (CPU_{exec\_cycles} + MemoryStallCycles) \times ClockCycle$$

$$MemoryStallCycles = IC \times \frac{MemAccess}{Instr} \times MissRate \times MissPenalty$$

#### 两级Cache平均访问时间

$$AMAT = HitTime_{L1} + MissRate_{L1} \times (HitTime_{L2} + MissRate_{L2} \times MissPenalty_{L2})$$

- $MissRate_{L2}$ 为**局部不命中率**（相对于到达L2的访问而言）
- **全局不命中率** = $MissRate_{L1} \times MissRate_{L2}$

#### Cache地址格式

$$\text{主存地址} = Tag + Index + BlockOffset$$

- $Index$ 位数 = $\log_2(Cache组数)$
- $BlockOffset$ 位数 = $\log_2(块大小)$
- $Tag$ 位数 = 主存地址总位数 - Index位数 - BlockOffset位数

#### 2:1 Cache经验规则

容量为N的直接映像Cache的不命中率 ≈ 容量为N/2的两路组相联Cache的不命中率

---

### 2.5 第6章 — I/O系统

#### 可靠性

$$FailureRate = \frac{1}{MTTF}$$

$$Availability = \frac{MTTF}{MTTF + MTTR}$$

- $MTTF$：平均失效前时间
- $MTTR$：平均修复时间

---

### 2.6 互联网络 — 互连函数

$N = 2^n$ 个结点的互联函数（输入/输出编号用 $n$ 位二进制表示 $x_{n-1}x_{n-2}...x_1x_0$）：

| 互连函数 | 变换 | 含义 |
|----------|------|------|
| 恒等置换 | $I(x) = x$ | 不变 |
| 交换置换 | $Exchange_k(x) = x \oplus 2^k$ | 第k位取反 |
| 立方体置换 | $Cube_k(x)$ | 同交换置换（第k位取反） |
| 混洗置换 | $Shuffle(x_{n-1}...x_1x_0) = x_{n-2}...x_1x_0x_{n-1}$ | 循环左移1位 |
| 反混洗 | $Shuffle^{-1}(x)$ | 循环右移1位 |
| 蝶式置换 | $Butterfly(x_{n-1}...x_1x_0) = x_0x_{n-2}...x_1x_{n-1}$ | 最高位与最低位交换 |
| PM2I函数 | $PM2_{+i}(x) = x + 2^i \pmod{N}$ | 加 $2^i$ 模N |
|  | $PM2_{-i}(x) = x - 2^i \pmod{N}$ | 减 $2^i$ 模N |
