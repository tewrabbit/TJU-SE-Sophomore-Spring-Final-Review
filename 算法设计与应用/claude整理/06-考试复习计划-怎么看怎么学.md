## 二、三阶段复习法

```
第1阶段：基础构建（占复习时间 40%）
    → 理解所有基本概念和算法思想
    → 阅读01、02号文档

第2阶段：强化训练（占复习时间 35%）
    → 掌握复杂度分析和解题技巧
    → 阅读03、04号文档 + 做作业题

第3阶段：冲刺巩固（占复习时间 25%）
    → 攻克DP和贪心两大难点
    → 阅读05号文档 + 模拟考试 + 回顾错题
```

## 三、每日详细计划

### 假设你有 7 天时间复习，分布如下：

---

### 📅 第1天：基础构建(一) —— 算法基础与复杂度

**上午（3小时）**：

1. ⏱️ **30分钟**：通读 `01-算法复习总纲` 的目录和第1-2章
2. ⏱️ **1.5小时**：精读第2章"算法效率分析基础"
   - 重点掌握：五个渐近符号的定义和性质
   - 重点掌握：非递归算法分析的5步法
   - 重点掌握：递归算法分析的5步法
   - 重点掌握：增长阶数排序（$1 < \log n < n < n\log n < n^2 < n^3 < 2^n < n! < n^n$）
3. ⏱️ **1小时**：打开 `02-核心算法专题突破`，阅读第1-3章（蛮力法、减治法、分治法的核心思想）

**下午（3小时）**：

1. ⏱️ **1.5小时**：打开 `03-算法复杂度分析完全指南`，精读：
   - 渐近符号的严格定义（O, Ω, Θ, o, ω）
   - 非递归分析例题（最大元素、元素唯一性、矩阵乘法）
   - 递归分析例题（阶乘、汉诺塔、斐波那契、折半查找）
2. ⏱️ **1.5小时**：主定理（Master Theorem）专项：
   - 理解 a, b, d 三个参数的含义
   - 比较 a 和 b^d 的三种情况
   - 用主定理分析：合并排序、折半查找、Karatsuba、Strassen

**晚上（1.5小时）**：
- ⏱️ **1.5小时**：打开 `04-考试题型专项训练`，做第五部分的复杂度分析题
  - 至少做完递推关系求解的3道大题
  - 至少做完渐近符号关系判断的4道小题
  - 做完极限证明题

## 四、怎么看PPT

### PPT的使用策略

PPT是**参考用**而非**主用**。因为你已经有了整理好的6份MD复习资料，它们已经融汇了PPT的所有精华。

**什么时候看PPT**：
- 某个概念的描述在MD中不够清晰时
- 想看某个算法的图示/流程图时
- 想确认老师的表述风格和术语时

**PPT重点章节优先级**：

| PPT | 重要性 | 原因 |
|-----|--------|------|
| Ch8 Backtracking and BB | 🌟🌟🌟 | 考试重点，图多，帮助理解解空间树 |
| Ch9 Dynamic Programming | 🌟🌟🌟 | 考试重点，DP的表和图很重要 |
| Ch5 Divide and Conquer | 🌟🌟🌟 | 核心章节 |

## 五、怎么做题

### 做题策略

**优先级排序**：
1. 🌟🌟🌟 自己的作业（hw1-hw3）：最贴近你的老师的出题风格
2. 🌟🌟🌟 期末考试真题：了解考试难度和题型
3. 🌟🌟 其他老师作业1-7：按主题分类的练习题
4. 🌟 其他老师作业2-10的学生答案：参考解题思路

### 做题原则

1. **先理解，再刷题**：不要在不理解算法思想的情况下盲目刷题
2. **手写伪代码**：考试要手写，练习时就手写
3. **分析复杂度**：每做一个算法题都同步分析其复杂度
4. **关注过程**：DP的填表过程、Dijkstra的dist数组变化过程、Prim的边选择过程——这些都是应用题考察的
5. **总结规律**：同类题目做完后总结解题模板

### 重要作业题目清单

基于对全部作业的分析，以下题目最值得做：

**复杂度分析**：
- 其他老师作业1：主定理 + 渐近符号判断题
- 其他老师作业2：Merge Sort伪代码 + 复杂度

**分治**：
- 自己的hw2：快排划分过程 + 分治/减治概念填空题
- 其他老师作业2：多项式除法 + 矩阵递归

**DP**：
- 自己的hw3：钢条切割 + 0-1背包 + 完全平方数 + 分割等和子集
- 其他老师作业3：股票交易（带冷静期）+ 编辑距离
- 其他老师作业4：矩阵链乘 + 双处理器调度 + 最大子段和(1D/2D/3D)

**贪心**：
- 其他老师作业5：Kruskal + 最大数排列

**回溯**：
- 其他老师作业6：圆排列问题 + 下一个排列
- 其他老师作业2：数独求解 + 分支限界背包

## 七、考试答题策略

### 时间分配建议

假设考试时间120分钟，满分100分：

| 题型 | 建议用时 | 策略 |
|------|----------|------|
| 选择题（5题5分） | 10分钟 | 快速判断，不会的先跳 |
| 简答题（5题30分） | 30分钟 | 每道6分钟，答要点即可 |
| 算法应用题（4题45分） | 55分钟 | 每道约14分钟，注意过程展示 |
| 复杂度分析题（1题20分） | 20分钟 | 认真计算，写出推导过程 |
| 检查 | 5分钟 | 检查伪代码语法、DP填表数字 |

### 答题细节提醒

**简答题**：
- 分点作答，结构清晰
- 概念定义 + 特征/步骤 + 对比/举例
- 不要只写关键词，要写完整句子

**算法应用题**：
- **先写算法名称和思路**
- **再写完整的伪代码**（按照伪代码规范.md的格式）
- 展示计算/推导过程（DP填表要画表，图算法要写每一步选择的边/顶点）
- 给出最终答案
- 标注时间复杂度

**复杂度分析题**：
- 严格遵循5步法
- 递推关系的推导过程要完整
- 主定理的应用要写出 a, b, d 的值和比较过程
- 最终结论用 $\Theta$ 符号表示（不要只用 O）

**选择题**：
- 注意"正确"vs"错误"vs"不正确"
- 注意"一定"vs"可能"
- 善用排除法
